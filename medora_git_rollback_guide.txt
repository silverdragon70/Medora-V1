Medora Git Rollback Guide

لو عايز ترجع للإصدار القديم (قبل AI):

1) الطريقة الآمنة (تنقل للنسخة القديمة بدون حذف):
git checkout before-ai-merge

2) الطريقة القوية (ترجع المشروع بالكامل لنقطة قديمة):
git reset --hard before-ai-merge

3) لو كنت رافع على GitHub وعايز توحده:
git push --force

مهم:
- استخدم reset بحذر لأنه بيمسح التعديلات الحالية محلياً
- الأفضل تبدأ بـ checkout للتجربة
